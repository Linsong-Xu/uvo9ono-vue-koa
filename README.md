# uvo9ono-vue-koa

 Community forum website written using vue and koa framework

## 安装Node

```shell
nvm install v10.16.0
```

## 安装MongoDB

```
cd mongotest
docker-compose up -d
```

## 安装Redis

```
cd redistest
docker-compose up -d
```
